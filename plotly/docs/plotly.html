 Description:
  Open-source JavaScript charting library behind Plotly and Dash
  --------------------------------------------------
  Plotly.newPlot(divID, data, layout, config)

    divID
        DOM node or string id of a DOM node
    data
        array of objects, see documentation
        (defaults to [])
    layout
        object, see documentation/
        (defaults to {})
    config
        object, see documentation
        (defaults to {})
 --------------------------------------------------
  The idea here is to provide functions which generate the code as strings.
  The low-level main function is plotly(data,layout,config,width,height).
  The Fricas draw functions are then based on it.
 Required: plotly-x.y.z.min.js
  There is no plotly-latest anymore, i.e. one has to update cdn if necessary
init : () -> STR 
 init() returns the CDN if not yet initialized, else "" is returned.
plotly : (STR,STR,STR,NNI,NNI) -> STR 
 plotly(data,layout,config,w,h) returns the html and javascript code
 as a string for setting up a unique 
 id with width w and height h
 and calling Plotly.newPlot(id,data,alyout,config).
displayHTML : STR -> Void() 
 Example: displayHTML draw(sin x, x=-4..4,[title("sin(x)")])
 Creates a plotly.id.html file and opens it in standard browser
