 Description: TMUX support.
tmux : String -> Void 
 tmux(cmd) executes a tmux command 'cmd' as if it were entered in the
 shell as 'tmux ...".
list_panes: () -> Void 
 list_panes() lists all panes (tmux lsp).
cmd_prompt: () -> Void 
 cmd_prompt() opens the tmux command prompt on the status line.
 (CTRL-B :)
display_msg : String -> Void 
 display_msg(msg) shows the message 'msg' in the status bar.
indicator : () -> Void 
 indicator() show the pane ids as a glimpse of big numbers.
find : String -> Void 
 find(s) ....
kill_session : () -> Void 
 kill_session() kills all sessions but the current.
send : (Integer,String) -> Void 
 send(id,cmd) sends 'cmd' to the pane 'id'.
